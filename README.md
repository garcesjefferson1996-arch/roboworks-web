# RoboWorks Web

Sitio web institucional de RoboWorks, escuela de robótica educativa para niños.

## Objetivo
Presentar los programas educativos de RoboWorks y facilitar la inscripción
mediante WhatsApp y formulario de contacto.

## Tecnologías
- HTML5
- CSS3
- Tailwind CSS
- JavaScript

## Estructura del proyecto
- index.html → Página principal
- css/ → Estilos personalizados
- js/ → Lógica JavaScript
- assets/ → Fuentes e iconos
- images/ → Imágenes del sitio
